# Quiz Application

A simple Quiz Application built with Python that asks multiple-choice questions, checks user answers, calculates the final score, percentage, and displays the performance grade.

## Features

- Multiple-choice questions (MCQs)
- Automatic score calculation
- Percentage calculation
- Performance grading
- Beginner-friendly code
- Command-line interface

## Technologies Used

- Python 3

## How to Run

1. Make sure Python is installed.
2. Download or clone this repository.
3. Open the project folder.
4. Run the following command:

```bash
python QuizApplication.py
```

- Python Basics
- Lists and Dictionaries
- Loops
- Conditional Statements
- User Input Handling
